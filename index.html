Costs

Costs

Costs é uma aplicação web de gerenciamento financeiro, projetada para ajudar empresas e pessoas a controlarem seus gastos e acompanharem suas finanças de forma eficiente. Com uma interface intuitiva, a aplicação permite a criação de orçamentos, o registro de gastos e a visualização de suas finanças em um só lugar.

A aplicação foi construída com React e utiliza o pacote react-router-dom para navegação fácil entre as páginas. Os dados são armazenados em formato JSON, garantindo uma experiência rápida e eficiente para o usuário. Com Costs, você tem acesso a uma solução prática e eficaz para gerenciar suas finanças.

Filmes_Lab

FilmesLab

Filmes_lib é uma plataforma web que apresenta cartazes de filmes e suas avaliações. Com ele, é possível descobrir novos filmes, ler resenhas e verificar suas classificações. Utilizando a API do TMDB, temos acesso a uma grande variedade de filmes e informações atualizadas.

O aplicativo foi desenvolvido com React, que é uma biblioteca JavaScript para a construção de interfaces de usuário. Além disso, foi utilizado o Vite para o gerenciamento de pacotes e otimização de desempenho. Juntos, estes dois frameworks garantem uma experiência de usuário rápida e fluida.

O projeto é perfeito para os amantes de filmes que desejam descobrir novas produções e ler avaliações antes de assistir. Além disso, é uma ótima opção para quem busca uma plataforma confiável e atualizada para obter informações sobre filmes. Com a API do TMDB, você pode ter certeza de que as informações apresentadas são precisas e atualizadas.

Secret Word

Secret Word

Secret Word é um jogo divertido e desafiante que coloca suas habilidades de dedução à prova. O objetivo é descobrir a palavra escondida através de uma dica e uma sequência de pontuação. Com a sua interface intuitiva, o jogo é fácil de jogar e ainda mais divertido de jogar com amigos e familiares.

Desenvolvido com React, Secret Word é rápido, responsivo e garante uma experiência de jogo suave. O código é bem escrito e organizado, garantindo que o jogo tenha uma boa performance. Além disso, o design é limpo e atrativo, tornando o jogo agradável aos olhos. Se você está procurando por uma forma divertida de passar o tempo e testar suas habilidades, o Secret Word é o jogo perfeito para você!

Bloco de Tarefas

Bloco de tarefas

O projeto Bloco de Tarefas é uma aplicação web desenvolvida com React, que tem como objetivo ajudar na organização de suas tarefas diárias. Ele permite que você anote suas tarefas, prevê o tempo que cada uma irá durar e mantém um registro se já foi concluída. Com a utilização do JSON Server, as informações são armazenadas de forma eficiente e acessíveis. Assim, você pode manter o controle de suas tarefas e garantir que elas sejam concluídas dentro do prazo.
